
# Reactjs-Navbar

NavBar in Reactjs with Tailwind css.

# Getting started

- RUN ``` npm install ```

Package to be installed:
- ``` react-router-dom ```

Packages to be imported:
- NavLink
- BrowserRouter
- Routes
- Route

Tailwind css installation command:
- ``` npm install -D tailwindcss ``` or visit "https://tailwindcss.com/"
